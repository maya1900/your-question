## 阶段 7：社区功能扩展

### 7.1 签到系统
- 设计签到数据模型（签到记录、连续签到天数）。
- 实现每日签到功能和积分奖励。
- 实现连续签到奖励机制。
- 添加签到日历展示。

### 7.2 通知系统
- 设计通知数据模型（通知类型、已读状态）。
- 实现系统通知（问题被回答、回答被采纳、获得点赞等）。
- 实现通知列表页面和未读标记。
- 添加通知中心入口和未读数量提示。

### 7.3 私信系统
- 设计私信数据模型（对话、消息）。
- 实现用户间私信发送和接收。
- 实现私信列表和对话详情页。
- 添加未读私信提示。

### 7.4 关注系统
- 设计关注关系数据模型。
- 实现关注/取消关注用户。
- 实现关注列表和粉丝列表。
- 在个人资料页显示关注数据。
- 实现关注动态流（可选）。

### 7.5 富文本编辑器
- 选择并集成富文本编辑器（推荐 Tiptap 或 Lexical）。
- 支持 Markdown 语法。
- 支持代码高亮。
- 支持图片上传（可选）。
- 替换问题和回答的纯文本输入框。

状态：待开发。

## 风险和处理

- Prisma 版本较新，迁移配置可能需要按实际 CLI 输出微调。
- Server Actions 和 cookie API 需要以当前 Next.js 版本为准验证。
- UI 从静态页迁移到真实数据后，需要检查空状态和长文本布局。
- 已补充核心业务自动化测试，页面级交互仍需手工验收。
